Place replacement LED Mask review videos in this folder.

The video carousel looks here first, then falls back to the current remote review videos if a file is missing.

Expected filenames:

- `review-01.mp4`
- `review-02.mp4`
- `review-03.mp4`
- Continue the same pattern up to `review-32.mp4`.

Use browser-friendly MP4 files, ideally H.264 video with AAC audio. The page plays them muted, looping, inline, and without visible controls.
